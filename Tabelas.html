Como utilizar tabelas no seu site

Tabelas podem ser extremamente úteis para mostrar diversas informações a seu usuário, desde valores de serviços a relações de dados específicos. O HTML nos proporciona uma maneira de implementá-las no nosso site de um modo simples, através de algumas tags e ajustes de informação.

Para utilizar tabelas, devemos implementar a tag <table>, a estrutura dela é semelhante a como listas funcionam, porém com algumas diferenças, no modo em que ela será exibida no site, variando conforme utilzá-mos suas tags. Abaixo há uma relação de cada tag que forma a tabela:

Tags de formação da tabela

<table>:
Como dito anteriormente, esta é a tag de formação da tabela, todas as outras tags serão implementadas dentro desta tag. Ela possui alguns atributos simples como width e border que podem ser utilizados para formatar de maneira simples a tabela sem utilizar CSS.
<tr>:
Esta é a tag responsável por criar linhas da tabela, as tags que formam as colunas estarão cituadas dentro desta tag.
<th>:
Esta tag é responsável pelo nome no topo de cada coluna, como um cabeçalho, esta tag somente é usada na primeira linha da coluna onde definimos os dados que serão inseridos em cada uma.
<td>:
Esta é a tag responsável por cada célula da tabela, sendo adicionada a cada tag de linha para conter uma informação, seja nome, idade, Carro preferido, etc. É importante adicionar ela na mesma ordem que foi estabelecida na primeira linha, ou seja, se o nome foi adicionado como primeira coluna, o primeiro valor a ser adicionado em cada linha é o nome, se a idade foi adicionada como oitavo valor na primeira linha, ele deve ser o oitavo valor a ser adicionado em todas as linhas.
