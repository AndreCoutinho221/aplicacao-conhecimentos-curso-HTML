Trabalhando com Listas em HTML

Listas podem não parecer grande coisa, mas são vitais para a construção de sites, elas estão presentes nos mais diversos locais, desde listas simples de compras até menus de navegação como o presente nesta e na maioria das páginas web, que apesar de estarem escondidos atrás de estilos CSS, ainda são listas.

Há 3 principais tipos de lista, as listas não ordenadas (como a do menu), listas ordenadas e listas de descrição, cada uma delas tem uma aplicação e utilização que se feita corretamente pode poupar muito tempo e esforço em seu site.

Listas não ordenadas

Listas não ordenadas como o nome ja diz, são listas que não tem uma ordem nos itens apresentados, no HTML padrão, elas possuem uma bolinha ao lado esquerdo para sinalizar cada item.

Para iniciarmos uma lista não ordenada usamos a tag <ul> que irá conter a outra tag da lista, a tag <li> que é reponsável por adicionar os itens na lista, cada uma destas tags representa um item dentro de uma lista, ela pode conter outras tags, como é feito no caso dos menus. Confira abaixo o código de uma lista não ordenada e logo após a aplicação dele.

<ul>
 <li>Item 1</li>
 <li>Item 2</li>
 <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

Listas ordenadas

Listas ordenadas são semelhantes as listas não ordenadas, porém como o nome ja diz, são listas com uma ordem. Elas são apresentadas em uma sequência de números, ou seja, o primeiro item terá o número 1, o segundo o número 2 e assim suscessivamente. A tag utilizada para formarmos uma lista ordenada é a tag <ol> que assim como a ul conterá as tags <li>. Há também alguns tipos de lista ordenadas, o padrão é o tipo de números, mas também existem os tipos de letras maiúsculas(A) e minúsculas(a), e romanas minúsculas(i) e maiúsculas(I), podemos alterar o tipo através do atributo type. Confira abaixo o código de uma lista ordenada e logo após a aplicação dele.

<ol>
 <li>Item 1</li>
 <li>Item 2</li>
 <li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Listas de definição

Listas de definição tem uma estrutura um pouco diferente das outras listas, elas são iniciadas pela tag <dl> e contém tags <dt> e tags <dd>. As tags <dt> são as tags que contém o que você quer definir, como um termo, uma palavra, um item, etc. Já a tag <dd> contém a definição do item, este tipo de lista foi muito utilizado na construção deste site, sendo perfeitamente visível na página de imagens. Cada tag <dt> precisa ser seguida de uma tag <dd>, para que a lista seja exibida da maneira correta. Confira abaixo o código de uma lista de definição e a aplicação do código logo abaixo.

<dl>
 <dt>Item 1</dt>
 <dd>Definição do Item 1</dd>
 <dt>Item 2</dt>
 <dd>Definição do Item 2</dd>
</dl>
Item 1
Definição do Item 1
Item 2
Definição do Item 2

Listas aninhadas

Listas aninhadas são listas dentro de listas, ou seja, uma lista não ordenada pode conter uma lista não ordenada, uma lista de descrição ou até mesmo uma outra lista ordenada. Confira abaixo algumas aplicações deste tipo de lista.

Lista ordenada dentro de uma lista ordenada:

<ol>
 <li>Item 1
  <ol>
   <li>Item 1.1</li>
   <li>Item 1.2</li>
   <li>Item 1.3</li>
  </ol>
 </li>
 <li>Item 2</li>
 <li>Item 3</li>
</ol>
  1. Item 1
    1. Item 1.1
    2. Item 1.2
    3. Item 1.3
  2. Item 2
  3. Item 3

Lista não ordenada dentro de uma lista ordenada:

<ol>
 <li>Item 1
  <ul>
   <li>Item 1.1</li>
   <li>Item 1.2</li>
   <li>Item 1.3</li>
  </ul>
 </li>
 <li>Item 2</li>
 <li>Item 3</li>
</ol>
  1. Item 1
    • Item 1.1
    • Item 1.2
    • Item 1.3
  2. Item 2
  3. Item 3

Lista não ordenada dentro de uma lista de definição:

<ol>
 <li>Item 1
  <dl>
   <dt>Item 1.1</dt>
   <dd>Definição do Item 1.1</dd>
   <dt>Item 1.2</dt>
   <dd>Definição do Item 1.2</dd>
  </dl>
 </li>
 <li>Item 2</li>
 <li>Item 3</li>
</ol>
  1. Item 1
    Item 1.1
    Definição do Item 1.1
    Item 1.2
    Definição do Item 1.2
  2. Item 2
  3. Item 3
