O que são Iframes e como utiliza-los em HTML

Iframes são uma maneira que o HTML nos disponibiliza para exibirmos páginas web dentro de outras páginas web, como se nosso site funcionasse como um navegador. Essa ferramenta nos disponibiliza muitas possibilidades, desde incorporação de vídeos externos até modos de personalização unicos de sua página web.

Apesar de ser uma tag bem surpreendente sua implementação é bem simples, utilizando apenas src, width e height já conseguimos implementá-los em nosso site, confira abaixo algumas aplicações possíveis para os Iframes.

Incorporando páginas externas

Como dito anteriormente, os Iframes permitem incorporar outros sites a nossa página, abaixo utilizamos a tag <iframe> fornecida pelo incorporar do youtube

Apesar de ser possível incorporar vídeos escrevendo o código, o youtube ja disponibiliza essa possibilidade se utilizarmos clicarmos em "Compartilhar" e após "Incorporar", forncendo a tag completa para uso no site

Outro uso além de vídeos e para a incorporação de endereços do google maps, que o próprio assim como o youtube fornece a tag já pronta para uso

Incorporando páginas do próprio site

Tendo em vista que podemos utilizar sites externos em nossa página, por que não poderiamos usar páginas internas? Basta trocar o src para apontar para uma página dentro do nosso site como foi feito logo abaixo

Interação dos Iframes com a tag <a>

Visto que podemos abrir páginas web nos Iframes, isso abre possibilidade de usarmos a tag de âncora para abrir links de sites no nosso site, funcionando como um navegador, para isso, precisamos adicionarmos um name ao nosso Iframe e passarmos ele através do atributo target da tag de âncora para ela
