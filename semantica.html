Semântica do HTML

Assim como outras linguagens, o HTML possui regras de escrita e de semântica para podermos utilizar a linguagem corretamente, há tags que devem ser utilizadas em determinados locais e há as que não podem ser utilizadas por não serem apropriadas para o uso naquele momento. Cada uma destas tags tem funções distintas que podem variar muito com base na pessoa que interpreta o conteúdo do seu site, que é por onde começamos a planejar a utilização das tags e onde elas devem e não devem ser usadas.

Estilização

Como dito acima, há tags que devem ser utilizadas para determinados fins e outras que não podem ser utilizadas, principalmente por não representarem o conteúdo que elas estão contento, grande parte do uso incorreto das tags de semântica se devem ao uso de tags apenas para estilização, que podem não representar o conteúdo que ela contém. A tag <div> não é uma tag semântica, logo ela não possuí nenhum significado quando escrita, porém ela é comumente utilizada para fazer estilizações no site, o que é o uso correto da tag, porém muitos sites acabam usando apenas esta tag para construir o site inteiro, o que é incorreto e não traz semântica e legibilidade ao código, tanto para desenvolvedores quanto para leitores de tela que não conseguem repassar corretamente o conteúdo aos usuários.

As tags semânticas não possuem uma regra exata para utilização como dito anteriormente, variando muito da interpretação do desenvolvedor de como utiliza-las corretamente, porém há algumas regras para utilização delas que devem ser levadas em consideração, um exemplo claro é a tag de navegação <nav> que deve ser utilizada para conter o menu de navegação principal do site, mesmo que uma tag div possa fazer as mesmas coisas que ela no sentido de estilização, ela não dirá aos desenvolvedores e a os leitores de tela que o que está dentro dela é o menu de navegação do site, o que não é exatamente ruim, mas pode causar alguns problemas no uso e na leitura do site.

Otimização para mecanismos de busca (SEO)

Os mecanismos de busca são um dos motivos principais que uso correto da semântica no seu site é importante, através do SEO conseguimos melhorar a classificação do site nesses buscadores e ganhar mais exibição. Claro que uma semântica correta não irá fazer milagres e trazer um conteúdo sem relevância para uma das primeiras páginas de busca, porém se combinarmos uma boa semântica com um conteúdo bem escrito e relevante conseguimos obter um resultado muito mais satisfatório e atrair mais usuários para seu site, gerando mais tráfego e consequentemente mais negócios.
