O que são links e como utilizá-los

Links nada mais são do que caminhos que existem na internet, ou melhor, endereços, cada endereço é responsável por uma página na internet e através destes endereços conseguimos navegar adequadamente na internet. Os links podem ser divididos em dois tipos, links relativos e links absolutos.

Links absolutos

Links absolutos são links que apontam para um endereço web específico, um endereço absoluto, são utilizados quando queremos anexar links de algum outro site ao nosso ou redirecionar o usuário a outro site. Um exemplo: Link para o Youtube, Link para o google

Links relativos

Links relativos são links que apontam para páginas dentro do seu site, ao invés de termos um link propriamente(como www.google.com.br) temos um caminho para um arquivo dentro do seu projeto que o levará a uma outra página, estes links estão presentes em todo site através de menus, botões e outras implementações. Um exemplo claro de implementação destes tipos de links é o menu deste site, que foi feito usando apenas links relativos.

Como implementa-los no seu site

Links são implementados através de uma tag de âncora(<a>), nesta tag possuímos o atributo "href" que é onde colocamos o link que irá marcar no texto.

Exemplo de link absoluto: <a href="https://www.google.com.br">Texto marcado como link</a>

Exemplo de link relativo: <a href="pasta-projeto/paginas/index.html">Texto marcado como link</a>
