O que são estilos CSS?

Conforme estudos realizados e a implementação deste site avançou, pude me deparar com estilos e a maneira que eles lidam e se juntam a outros elementos para compor uma mensagem que o designer/programador quer passar, os estilos apesar de não ser uma parte crítica para a mensagem, servem como uma maneira de passar sua mensagem, melhorar a leitura, usabilidade entre outros aspectos do site em questão, podendo tanto fazer de um site com pouco conteúdo um panfleto perfeito, mas também transformar um excelente conteúdo em uma atrocidade visual.

Os estilos CSS podem ser feitos de diversas formas, o mais popular nos tempos atuais é usar um framework css tailwind, onde compilamos um arquivo CSS com base nos estilos usados no HTML, onde já temos as classes de maneira mais simples e correta, porém sites mais antigos podem usar implemenações mais simples como a deste site, onde uma folha de estilos CSS é adicionada manualmente e os estilos são feitos pelo programador. Também há várias convenções sobre nomes de classes e modularização de estilos, dentre eles eu dou um destaque a BEM, a qual eu tentei seguir neste site, mas sem muito sucesso.

Frameworks

Como mencionado anteriormente, os estilos CSS podem ser usados através de frameworks, onde podem ter várias implementações, como o tailwind mensionado acima, mas também com diversas outras como o superCSS, onde compilamos vários arquivos CSS e transformamos em um só, facilitando o desenvolvimento por parte dos programadores.

Em sua maioria, estes frameworks vão usar como base o JavaScript, que também é uma ferramenta maravilhosa quando se trata de estilização, responsividade e até mesmo implementações maiores e mais sofisticadas, como um servidor inteiro, sendo a principal ferramenta para desenvolvedores front-end por ser uma linguagem de compreenção simples e com muitas implementações possíveis.
